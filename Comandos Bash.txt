Comandos Bash

ls - lista tudo do diretório | ls -a - mostra arquivos ocultos
cd - entra em um diretório | cd .. - sobe para a pasta acima
pwd - mostra o caminho absoluto da navegação
mkdir - cria um diretório
cat - mostra o que tem dentro do arquivo
git init - transforma a pasta atual em repositório / pasta versionada
rm -rf - remove de forma recursiva (tudo do diretório)
rm - remove um arquivo
touch <nome_do_arquivo> - cria um arquivo vazio (ex: touch README.md)
